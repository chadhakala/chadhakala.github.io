# Open Sores Homepage Prototype

Static homepage prototype for early visual direction and stakeholder feedback.

## Structure

- `index.html` — homepage
- `styles/` — base and page styles
- `scripts/` — light interactions
- `assets/` — images and media

## Workflow

- Build on feature branches
- Open pull requests for review
- Merge approved changes into `main`
